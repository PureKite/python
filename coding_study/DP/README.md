## 다이나믹 프로그래밍
-----
### ❓ 다이나믹 프로그래밍이란?
- 다이나믹 프로그래밍을 사용할 수 있는 조건
  1. 큰 문제를 작은 문제로 나눌 수 있다.
  2. 작은 문제에서 구한 정답은 그것을 포함하는 큰 문제에서도 동일하다.
- 탑다운(메모이제이션) 방식, 하향식 : 큰 문제를 해결하기 위해 작은 문제를 호출
- 보텀엄 방식, 상향식 : 단순히 반복문을 이용하여 소스코드를 작성하는 경우 작은 문제부터 차근차근 답을 도출
